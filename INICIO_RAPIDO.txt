🚀 INICIO RÁPIDO

1️⃣  BACKEND - Compilar y Ejecutar
   ────────────────────────────────
   $ cd biblioteca
   $ ./mvnw clean install
   $ ./mvnw spring-boot:run
   
   ✅ Backend corriendo en: http://localhost:8080

2️⃣  FRONTEND - Servir Archivos
   ──────────────────────────────
   Opción A: Live Server (VS Code)
     • Click derecho en carpeta frontend/
     • "Open with Live Server"
   
   Opción B: Python
     $ cd frontend
     $ python -m http.server 8000
   
   ✅ Frontend disponible en: http://localhost:8000

3️⃣  ACCEDER
   ─────────
   Login: http://localhost:8000/ingreso/index.html

❓ PROBLEMAS COMUNES

1. "Puerto 8080 ya está en uso"
   → Cambiar en application.properties: server.port=8081

2. "No puedo conectar al backend"
   → Verificar que está corriendo en localhost:8080
   → Verificar que la BD MySQL está disponible

3. "Usuario no encontrado"
   → Crear usuarios de prueba (ver README.md)
   → Usar "admin" / "admin123"
